# VoiceGo — Hướng dẫn sử dụng

VoiceGo là app **đặt xe bằng giọng nói cho người khiếm thị / khuyết tật vận động** ở TP.HCM. Mọi thao tác chính bằng giọng nói; nút bấm chỉ là phụ trợ.

> 🔗 **Demo (chạy local):** https://10.236.55.216:5173/ — cách vào qua cảnh báo chứng chỉ + yêu cầu **cùng WiFi** xem [DEMO.md](DEMO.md).

## Đăng nhập
Mở app → nhập email + mật khẩu (tài khoản demo ở [SETUP.md](SETUP.md)). Vai trò (khách / tài xế) tự nhận theo tài khoản.

## A. Khách hàng (người khiếm thị)
1. **Trang chủ** → chạm **“Đặt chuyến bằng giọng nói”** (hoặc ô “Xe máy” / “Ô tô”).
2. Vào màn trợ lý: **tự chào và tự bật micro**. Cứ **nói điểm đến** (vd “tôi muốn đến chợ Bến Thành”).
   - Nếu nơi đến có **nhiều cơ sở / nhiều quận** (vd “bách khoa”, “19 Ngô Gia Tự”) → agent **đọc danh sách** và hiện các điểm **trên bản đồ**; nói **số thứ tự** hoặc tên/khu vực để chọn.
   - Với địa điểm có **cổng dễ tiếp cận** (vd ĐH Bách Khoa cơ sở 1) → agent gợi ý **Cổng 1 (dễ tiếp cận)**; điểm này tô **màu xanh** trên bản đồ.
3. Agent hỏi **“xe máy hay ô tô?”** → trả lời. Nếu trả lời mơ hồ (“đoán đi”, “tùy”) agent sẽ **hỏi lại** chứ không tự chọn.
4. Agent **đọc giá + quãng đường + thời gian** → nói **“đồng ý”** để đặt (hoặc “đổi xe” / “đổi điểm đến”).
5. **“Đang tìm tài xế…”** → khi tài xế nhận: agent đọc **tên + biển số + thời gian tới**.
6. Tài xế đến nơi đón: agent **đọc mã PIN 2 lần** + hiện PIN cỡ lớn. **Đọc mã PIN cho tài xế** để xác minh an toàn.
7. Xác minh xong → **“Đang di chuyển”** (chạm thanh tay cầm để thu gọn/mở rộng, xem bản đồ) → **“Đã tới nơi”**: có thể **đánh giá mức độ tiếp cận** (sao + lối vào) để giúp cộng đồng.

> Nút micro tròn dưới cùng: chạm để **tắt** nghe (tắt hẳn) hoặc **bật lại** (vẫn nhớ hội thoại). Nói **“huỷ”** bất kỳ lúc nào để dừng.

## B. Tài xế
1. Đăng nhập tài khoản tài xế → màn **“Đang chờ cuốc xe”**.
2. Có khách → hiện **“Có cuốc xe mới”** kèm badge **“♿ Chuyến chở người khuyết tật”** + ghi chú hỗ trợ → bấm **“Nhận cuốc”**.
3. Đang đến đón → bấm **“Tôi đã đến nơi”** → màn nhập **mã PIN**.
4. Hỏi khách đọc mã PIN → **nhập đúng** → xác minh an toàn → bấm **“Hoàn thành chuyến đi”**.
